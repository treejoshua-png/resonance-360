# resonance-360

Resonance-branded MVP blueprint and interactive prototype for an executive 360 feedback + AI coaching tool.

## Included in this commit
- Interactive front-end prototype (`app/`) with:
  - weighted multi-rater score table,
  - self overlay comparison,
  - dual radar chart,
  - report output mapping,
  - AI boundary block.
- Product and technical blueprint docs (`docs/`).

## Run locally
```bash
cd app
python3 -m http.server 4173
```
Open `http://localhost:4173`.

## Notes
This is an MVP foundation for rapid iteration toward full workflow automation (invites, reminder emails, report export, and secure AI coaching integration).
